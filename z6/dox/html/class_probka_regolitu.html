Detailed Description

Klasa ProbkaRegolitu.

Modeluje pojęcie problki regolitu czyli materiału zbieranego przez laziki z powierzchni marsa. Probka regolitu to obiekt geometryczny więc, klasa dziedziczy po ObiekGeom

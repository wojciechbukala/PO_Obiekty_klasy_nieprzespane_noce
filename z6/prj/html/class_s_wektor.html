Detailed Description

template<typename typ, int wymiar>
class SWektor< typ, wymiar >

Szablon klasy modelującej pojęcie wektora. Klasa ta jest szablonem i może stworzyć wektory dowolnie dużej wielkości i zawierający elementy różych typów.

Parameters
[in]typ- typ zmennych składowych wektora (np. double, int, LZespolona).
[in]wymiar- rozmiar wektora, liczba całkowita int.

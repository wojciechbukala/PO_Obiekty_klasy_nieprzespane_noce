Detailed Description

template<typename typ, int wymiar>
class SMacierz< typ, wymiar >

Szblon klasy Macierz.

Szablon klasy modelującej pojęcie macierzy. typ - typ zmennych składowych macierzy (np. double, int, LZespolona). wymiar - rozmiar macierzy kwadratowej, liczba całkowita int.

Parameters
[in]typ- typ elementow macierzy (np int, double, Wektor).
[in]wymiar- wymiar macierzy kwadratowej.

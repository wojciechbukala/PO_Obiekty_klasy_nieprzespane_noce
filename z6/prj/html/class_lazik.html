Detailed Description

Klasa Lazik.

Dziedziczy publicznie do klasie ObiektGeom. Modeluje pojęcie łazika będącego jednym z obiektów geometrycznych. Posiada metody odpowiedzialne za rotacje oraz jazde łazika.
